# ADR Storage Convention

All Architectural Decision Records (ADRs) are stored in the `docs/adr/` directory.

## File Format
- Each ADR is a Markdown file.
- Files use YAML frontmatter to include metadata (id, projectId, title, createdAt).

## Purpose
ADRs document the "why" behind significant design decisions, providing a historical context that helps both humans and AI agents understand the evolution of the architecture.
